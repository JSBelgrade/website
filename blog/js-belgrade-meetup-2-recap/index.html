JS Belgrade Meetup

Posted by stojanovic Jan 0th 2015

U subotu 10. januara u prostorijama Cloud Horizona održan je drugi JS Belgrade Meetup kojem je prisustvovalo 14 osoba.

Tema ovog meetupa je bila "Browserify i Webpack", ali smo pored toga pričali i o Reactu, NodeJSu i io.js, testiranju i mnogim drugim temama (uglavnom) vezanim za JavaScript.

Prezentaciju Gorana Gajića o Webpacku možete videti ovde: http://slides.com/feroc1ty/webpack. Bogdan je o Browserify pričao uz live code primer.

Naredni JS Belgrade Meetup će biti održan u subotu 14. februara, a tema će biti testiranje JavaScript koda.
