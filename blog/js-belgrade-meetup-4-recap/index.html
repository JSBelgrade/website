JS Belgrade meetup

Posted by stojanovic Mar 0th 2015

U subotu 21. marta u prostorijama Cloud Horizona održan je četvrti JS Belgrade Meetup na kome je prisustvovalo 18 osoba.

Tema ovog meetupa je bio React, pričali smo o virtuelnom DOMu, JSXu, problemu koji React rešava i sl. Dodatna diskusija o virtuelnom DOMu ostaje za neki naredni meetup.
